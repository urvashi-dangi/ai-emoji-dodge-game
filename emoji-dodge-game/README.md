# 🎮 Emoji Dodge (Flutter Game)

Emoji Dodge is a simple arcade-style mobile game built using Flutter.

## 🕹️ Gameplay
- The player controls a character at the bottom of the screen.
- Emojis fall from the top continuously.
- Avoid collisions to survive as long as possible.
- Score increases over time.

---

## 🚀 Features
- Smooth player movement (drag controls)
- Falling emoji obstacles
- Collision detection
- Score tracking
- Game Over dialog with restart option
- Increasing difficulty over time

---

## 📱 Screens
1. Home Screen
    - Game title

2. Game Screen
